## 開發者指引

若要使用 QFPay OpenAPI，你必須先取得一組有效的 API 憑證，包括：

- `app_code`
- `client_key`
- （選填）`mchid`，若你的商戶結構需要此參數

<Note>
若 QFPay 有提供 `mchid` 給你，則你在每一次 API 請求中都必須提交 `mchid`（除非另有說明）。  
若 **未提供** `mchid`，請 **不要** 在請求中包含此參數。
</Note>

### 憑證與商戶結構

- 每位商戶會獲得唯一的 `app_code` 與 `client_key`。
- 若商戶有多間分店，還會額外獲得 `mchid`，用以區分不同門市。
- `mchid` 是一組加密字串，用於識別特定的分店或門市。

### 編碼格式

除非另有說明，所有的請求與回應資料皆使用 `UTF-8` 編碼。
