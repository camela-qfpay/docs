## 支付方式說明

`pay_type` 參數指定了本次交易使用的支付渠道，例如支付寶、微信支付或信用卡通道等。該參數為必要欄位，必須正確填寫，否則將導致交易失敗。

此參數直接影響交易的路由方式、介面交互方式（例如是否需要掃碼、喚起原生錢包等）以及授權流程。

### 使用位置

在調用 `/trade/v1/payment` 或其他支付相關 API 時，請於請求資料中加入 `pay_type` 欄位。例如：

```json
{
  "pay_type": "800101",
  "txamt": "1000",
  ...
}
```

:::note
非所有商戶都開通所有 pay_type。請參考你的開通資料，或聯繫 technical.support@qfpay.com 確認。
:::

## 特別備註

- **801101**: 此支付方式要求交易金額 **必須高於 1 港元**。

- **802001**:  此支付方式  **不支援退款**.
