# 支援的交易貨幣

本節列出 QFPay API 可接受的貨幣代碼。提交交易時，`txcurrcd` 參數的值必須為下列其中一種。

所有貨幣代碼均遵循 ISO 4217 國際標準（3 個大寫英文字母）。

| 貨幣代碼 | 貨幣名稱 |
|---------|-----------|
| AED     | 阿拉伯聯合大公國迪拉姆 |
| CNY     | 人民幣 |
| EUR     | 歐元 |
| HKD     | 港元 |
| IDR     | 印尼盾 |
| JPY     | 日圓 |
| MMK     | 緬甸元 |
| MYR     | 馬來西亞令吉 |
| SGD     | 新加坡元 |
| THB     | 泰銖 |
| USD     | 美元 |
| CAD     | 加拿大元 |
| AUD     | 澳元 |

:::note
部分支付方式可能只支援 **HKD**（例如 FPS、AlipayHK）。在進行非港元交易前，請務必與你的整合負責人確認支援情況。
:::
